About Us

RedStore is a fictional project I started by following a YouTube tutorial to improve my web development skills. Through this project, I’ve gained a much better understanding of HTML, CSS, and JavaScript, particularly in building responsive websites. I’ve learned how to structure a website with clean, reusable code, style pages with CSS, and add interactive features like a toggle menu using JavaScript. I also had to troubleshoot some issues with the layout and navigation, which taught me how to break down problems and research solutions effectively. One challenge was ensuring the site looked good across different devices, which helped strengthen my problem-solving and debugging abilities. While I’ve learned a lot from this project—such as using external libraries like Font Awesome—it still has a lot of work left to finish, and I look forward to refining it further as I continue developing my skills.

Our Mission
